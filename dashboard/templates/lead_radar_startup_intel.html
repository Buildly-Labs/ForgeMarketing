API Key Setup

The sources below require free API keys. Add them in Admin → System Config → Startup Intel Plugin.

Product Hunt API v2

New product launches — great signal for "just shipped, likely founder-led".

producthunt.com/v2/oauth/applications →

Create an application → copy the "Developer Token". Store as startup_intel_product_hunt_token.

OpenCorporates

Recently incorporated US companies. Pairs well with SEC Form D filings.

opencorporates.com/api_accounts/new →

Register for a free account. Store key as startup_intel_opencorporates_key.

Companies House (UK)

Newly formed UK companies — free official government data including officers.

developer.company-information.service.gov.uk →

Register, create an application. Store key as startup_intel_companies_house_key.

GitHub Token (optional)

Not required, but raises enrichment rate limit from 60 to 5,000 requests/hour.

github.com/settings/tokens/new →

No scopes needed for public data. Store as startup_intel_github_token.

Free sources (no key needed): YCombinator directory (updated daily by YC) · SBIR/STTR awards (sbir.gov) · NSF Awards (research.gov) · SEC Form D filings (sec.gov) · Hacker News "Launch HN" and "Who is Hiring" posts
